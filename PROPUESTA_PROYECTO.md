# Propuesta y Finalidad del Proyecto

## Nombre del proyecto
**Centro de Alertas Marinas**

## Propuesta
Desarrollar una aplicación web interactiva para el monitoreo de condiciones oceánicas en tiempo casi real, enfocada en variables críticas para seguridad y operación marítima:

- Temperatura superficial del mar.
- Altura de oleaje.
- Estado de alerta por severidad (vigilancia, alerta y peligro).
- Consulta puntual de condiciones marinas sobre el mapa.
- Línea de tiempo de 24 horas para analizar evolución temporal.

La plataforma se implementa con `React + Vite + TypeScript` y `MapLibre`, incorporando visualización geoespacial, capas dinámicas y datos de `Open-Meteo Marine API`.

## Finalidad
La finalidad principal es **mejorar la toma de decisiones** ante riesgos marinos mediante una visualización clara, rápida y accionable de las condiciones del mar.

Este proyecto busca:

1. **Prevenir riesgos** para navegación, pesca y actividades costeras.
2. **Anticipar eventos críticos** mediante alertas por umbrales.
3. **Facilitar análisis temporal** con el historial reciente de condiciones.
4. **Acercar datos técnicos** a usuarios no especialistas mediante una interfaz simple.

## Problema que resuelve
Actualmente, la información marina suele estar dispersa en múltiples fuentes y formatos técnicos. Esto dificulta:

- Interpretar rápidamente el nivel de riesgo en una zona.
- Comparar cambios por hora de manera visual.
- Tomar decisiones operativas con contexto geográfico claro.

La solución unifica estas necesidades en una sola vista de mapa con controles mínimos y lectura inmediata.

## Usuarios objetivo
- Navegantes y embarcaciones menores.
- Operadores de turismo marítimo y deportes acuáticos.
- Comunidades pesqueras.
- Equipos de gestión del riesgo y monitoreo costero.

## Alcance funcional actual
- Mapa interactivo con capas de temperatura, oleaje y corrientes.
- Alertas por estaciones marinas con clasificación de severidad.
- Geolocalización del usuario con enfoque de 100 km.
- Línea de tiempo (24 h) para explorar cambios por hora.
- Popups de consulta marina con datos consistentes a la hora seleccionada.
- Paneles y leyendas optimizados para una experiencia visual limpia.

## Valor esperado
- Reducción del tiempo de interpretación de condiciones marinas.
- Mayor conciencia situacional para decisiones en campo.
- Base tecnológica escalable para integrar nuevas fuentes, modelos y notificaciones.

## Visión de evolución
A mediano plazo, el proyecto puede incorporar:

- Predicción de riesgo por zona (modelos estadísticos/ML).
- Notificaciones automáticas por geocerca y nivel de alerta.
- Integración con datos satelitales y boyas en tiempo real.
- Reportes históricos por región y exportación de análisis.
